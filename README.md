# 宠物商城

### 项目简介
一个出售宠物和宠物用品的电子商城。

### 效果图
![image](http://pp5v53szm.bkt.clouddn.com/pets-shopping.png)

### 技术栈
- Vue Cli3 脚手架
- Vue全家桶

### 特性
- 登录/登出
- 购物车
  - 加入购物车
  - 查看详情
  - 结算
- 用户
  - 订单管理
  - 博客管理
  - 设置
